# ワークショップメーカー 市場調査 & 営業戦略レポート

**調査日：2026年3月28日**
**対象アプリ：** https://workshop-project-mocha.vercel.app/

---

## アプリ概要

- **機能：** ワークショップ自動設計ツール
- **仕組み：** 参加者数・目的・レベル・時間を入力するだけでAIが最適プログラムを自動生成
- **ライブラリ：** アイスブレイク30種以上・メインワーク（チームビルディング/コミュニケーション/リーダーシップ等）・クロージング11種
- **特徴：** 演劇的手法を活用した実践的なワークショップ設計

## 狙うべき市場セグメント（優先順位順）

### 優先度 S：フリーランス研修講師・ファシリテーター

- **市場規模：** FAJコア会員1,600名 ＋ 周辺層（研修担当者等）数万人規模
- **ニーズ：** 1人で複数クライアントを掛け持ち → 設計業務の効率化が死活問題
- **刺さる価値：** 「3時間かかってたプログラム設計が数分で完成」
- **獲得戦略：** FAJ（日本ファシリテーション協会）、ストアカ講師コミュニティへの浸透

### 優先度 A：中堅・中小企業の人事担当者（兼務層）

- **市場規模：** OFF-JT実施事業所73.8%・非専任担当者が大多数
- **ニーズ：** 専門知識なしでプログラムを作らなければならない
- **刺さる価値：** 「研修設計の専門知識ゼロでもOK」
- **獲得戦略：** HRテック展示会、SmartHR/HRBrainなどHRSaaS連携

### 優先度 A：インプロ・演劇系ファシリテーター（アーリーアダプター層）

- **市場規模：** 急拡大中（メルカリ・中外製薬等大手が採用）
- **ニーズ：** 演劇的手法のライブラリ＋設計ツールが一体化したものが存在しない
- **刺さる価値：** 競合が完全に存在しないジャンル
- **獲得戦略：** インプロアカデミー・ミテモ等との連携、演劇×ビジネス系コミュニティ

### 優先度 B：大企業・上場企業の人材開発部門

- **市場規模：** 最大の予算保有層
- **ニーズ：** 人的資本経営の開示義務化で研修の質・量の可視化が急務
- **刺さる価値：** 設計の標準化・スケールアップ
- **獲得戦略：** 最初は個人ユーザー経由でボトムアップ浸透、後にエンタープライズ展開

### 優先度 C：教育機関・NPO

- アクティブラーニング・PBL導入校、まちづくり団体、市民活動組織
- 予算は低いが口コミ拡散力が高く、早期採用者として有効

## 競合マップと差別化

| サービス | 特徴 | 本アプリとの差 |
|---------|------|--------------|
| SessionLab（海外） | 設計補助ツール・1,000種アクティビティ | **日本語なし・AI自動生成なし** |
| Butter.us（海外） | ライブ進行管理 | 設計自動化機能なし |
| Miro / MURAL | 汎用ホワイトボード | ワークショップ特化でない |
| インソース・ALL DIFFERENT | 研修会社（受注型） | SaaSではなくサービス |

**結論：日本語対応のワークショップ自動設計AIはほぼ存在しない。ブルーオーシャン。**

## 営業戦略（フェーズ別）

### Phase 1：種まき期（0〜6ヶ月）— 無料で拡散

**目標：** コアファン100人獲得・口コミ基盤構築

| アクション | 詳細 |
|-----------|------|
| **コミュニティ浸透** | FAJ（日本ファシリテーション協会）の月例会・勉強会で無料デモ |
| **ストアカ講師への直接アプローチ** | ファシリテーション系人気講師にDM → 無料プロアカウント提供 |
| **SNS発信** | 「ワークショップ設計が3分でできた」体験動画をX/note/LinkedInで拡散 |
| **インプロコミュニティ** | インプロアカデミー・ミテモ系ファシリテーターへのアプローチ |
| **noteでの情報発信** | 「アイスブレイク30選」「チームビルディング設計のコツ」など有益コンテンツ |

### Phase 2：マネタイズ期（6〜12ヶ月）— 課金モデル構築

**推奨料金体系：**

| プラン | 対象 | 価格 | 内容 |
|--------|------|------|------|
| Free | 個人・お試し | ¥0 | 月3プログラムまで生成 |
| Pro | フリーランス講師 | **¥2,980/月** | 無制限生成・PDF出力・カスタマイズ |
| Team | 中小企業人事 | **¥9,800/月** | 複数ユーザー・テンプレート共有 |
| Enterprise | 大企業・研修会社 | 要見積もり | API連携・管理画面・カスタムライブラリ |

### Phase 3：BtoB拡大期（12ヶ月以降）— 法人営業

| チャネル | アクション |
|---------|-----------|
| **HR展示会** | Japan HR Challenge、HRエキスポへの出展 |
| **研修会社OEM提携** | インソース・ミテモ等へのOEM・API提供 |
| **HRSaaS連携** | SmartHR・HRBrainなどからの流入導線構築 |
| **大企業ボトムアップ** | 個人Pro利用者が社内展開→Team/Enterprise転換 |

## アクション優先順位（今すぐできること）

1. **FAJの次回月例会にスピーカー/デモ枠で申し込む** — 最速でコアターゲットにリーチ
2. **ストアカでファシリテーション講座を持つ人気講師トップ20人にDM** — 無料Proアカウントを提供してアンバサダー化
3. **noteで「ワークショップ設計の教科書」連載開始** — SEO流入とブランド構築
4. **X（Twitter）で「3分で作ったワークショッププログラム」実演投稿** — 拡散起点
5. **インプロ系企業（ミテモ・インプロアカデミー）に提携打診** — 演劇的手法コンテンツとの相乗効果

## 総括

最大のチャンスは「**日本語 × AI自動生成 × 演劇的手法**」の三位一体の組み合わせが競合ゼロであること。

まずフリーランスのファシリテーターをファンにして、口コミで企業人事層へ浸透させる**ボトムアップ戦略**が最も再現性が高い。
